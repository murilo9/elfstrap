Navbars

- A classe elfnav permite que uma div seja transformada facilmente em uma navbar. Para inserir os links em uma nav bar basta adicionar listas não-ordenadas que o estilo é aplicado automaticamente (sem necessidade de acrescentar nenhuma outra classe).

- Um elmento li de uma lista não-ordenada que tenha a id active recebe um destaque especial.

- Um form pode ser inserido dentro da navbar com uma input do tipo text, que ele será estilizada automaticamente.
