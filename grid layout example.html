Grid Layout

- O sistema de grid layout responsivo do elfstrap funciona em qualquer elemento (preferencialmente divs) que possuir a classe col-x (onde x é o tamanho, podendo ser 1, 2, 3, 4, 6, 8, 10 ou 12) e estiver dentro de um elemento com a classe row. O somatório do tamanho das colunas col-x não deve ultrapassar 12.
